# zmq
Lightweight broker using ZMQ, allowing messaging across the network

Set up
1. Start the Broker 
python broker.py

2. Start a publisher
python testqos.py

3. Start a subscriber
python testenbodeb.py

4. Start a logger
python logger.py

in any particular order.
